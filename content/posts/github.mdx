### Github 项目推荐
GitHub是全球最大的程序员交友平台，里面有大量优质的开源项目，例如比较著名的有：ffmpeg,chatgpt-next-web,motirx 

里面也有一些特别适用的，也有些搞笑的，在这里统一做个人推荐（此文章持续更新，不要错过哦）

#### Snap.Hutao胡桃工具箱
![项目图片](https://image.wujiyan.cc/imgs/2024/11/28/67486c7e08e3c.jpeg)
实用原神工具箱，让你快速知道原神直播原石兑换码等非常有用的消息，还支持一键启动，了解个人角色的培养，你值得拥有！
官网：https://hut.ao/zh

#### ToastFish 通知栏背单词内卷神器
![项目图片](https://image.wujiyan.cc/imgs/2024/11/28/67486c8d74107.jpeg)
star 数 5000！相当恐怖！这个项目用 Windows 原生通知来显示单词，充分利用空闲时间，游戏死亡等待重生时的好伙伴，这样一局游戏下来，起码能记住 1-2 个，真正的内卷神器！

GitHub 下载（官方仓库 Releases）：https://github.com/Uahh/ToastFish/releases
